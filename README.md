# Vanderperre_et_al
FIJI Java scripts to quantify fluorescent signals throughout the study:
* Macro_Coloc.ijm = measures colocalization between ubx and FISH signals within nuclei
* Macro_DensityMaps.ijm = creates density maps from an ensemble of nuclei analysed for colocalization
* Macro_Distances.ijm = measures 3D distances between a nuclear FISH signal and the nuclear enveloppe
* Macro_Distrib.ijm = measures Ubx signal distribution inside nuclei, in concentric regions from the periphery inward
